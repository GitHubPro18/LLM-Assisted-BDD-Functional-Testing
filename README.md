# LLM-Assisted BDD Functional Testing Platform

This project implements a governance-first, LLM-assisted BDD testing system as described in the PRD.

## Folder Structure

- requirements/: Business requirements input
- llm/: LLM scenario generation
- validation/: Scenario validation
- approval/: Manual approval gate
- bdd/: BDD features, steps, and config
- execution/: Test execution
- reports/: All generated reports

See PRD for full details.
